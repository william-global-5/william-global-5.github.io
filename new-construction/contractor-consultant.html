Contractor & Consultants

The FDOT Traffic Engineering and Operations Office keeps Florida safely on the go. For example, the ITSFM software enables accurate data and camera feeds to the statewide 511 Traffic Information System, used by millions of people each year. ITSFM helps keep FDOT’s real-time information accurate.

The contractor is responsible to deliver as-built documentation for the complete traffic control signal and device installation upon its completion. Standard Specifications Section 611, “Acceptance Procedures for Traffic Control Signals and Devices,” January 2017 has expanded the as-built for new construction projects to include modifications to the as-built plans and equipment inventory data necessary to support the Department’s ITS Facility Management (ITSFM) system.

Key changes to Section 611 include:

  • The addition of Site Identification Names to all support structures, equipment cabinets, and other fixed location features,
  • Assigning a unique Site Identification Name to create associations between the as-built plans, inventory forms, and the ITSFM system
  • Feature Import Templates:
    • Submit as-built data using Feature Import Templates for use in the Department’s ITSFM system.
    • Include data for all the components listed in components except for:
      • Loops and Detection Zones
      • Signal Heads
    • Fiber Optic Cables:
      • Path names
      • Cable manufacture, type, and strand count and other attributes
      • Cable sequential reading:
        • Each pull box entrance / exit,
        • Each side of the splice enclosure, and
        • Patch panels

FDOT understand the effort required of Contractors and Consultants to produce quality as-built documentation. Recognizing this, the Department developed ITSFM data collection forms, custom GPS mapping configuration, data dictionary and export routines, and feature import template tools designed to both standardize and simplify the data collection process. These tools are synchronized with the ITSFM database and available to contractor and consultants for use in preparing as-built deliverables.
