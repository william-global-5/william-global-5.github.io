The FDOT Traffic Engineering and Operations Office keeps Florida safely on the go. For example, the ITSFM software enables accurate data and camera feeds to the statewide 511 Traffic Information System, used by millions of people each year. ITSFM helps keep FDOT’s real-time information accurate.

FDOT’s as-built documentation requires all support structures, equipment cabinets, pullboxes and other features with a coordinate to be assigned a unique Site Identification Name that includes:

  • Device type
  • Route
  • nearest 1/10 mile marker
  • Nearest travel lane
  • Unique identifier

FDOT will furnish GPS seed files containing the 1/10 mile marker locations along each route to serve as the baseline for as-built mapping. The seed file includes the ITSFM data dictionary that can be updated to include project specific road names when the names are provided by the contractor / consultant. To request GPS Seed Files, email the project number, contact information, route name, start and end mile marker location to gps-seed-files@fdot.gov.
