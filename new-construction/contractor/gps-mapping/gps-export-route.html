Traffic Engineering and Operations Office

TEO / Divisions / TSMO / TPM / ITSFM / Contractor & Consultants

GPS Export Route

The FDOT Traffic Engineering and Operations Office keeps Florida safely on the go. For example, the ITSFM software enables accurate data and camera feeds to the statewide 511 Traffic Information System, used by millions of people each year. ITSFM helps keep FDOT’s real-time information accurate.

FDOT developed custom ITSFM Specific Export Configurations for use with Trimble’s Pathfinder Office software. The following export options are available:

Description this export outputs feature and attribute description values (i.e. Fiber Pullbox).
ITSFM Code this export outputs ITSFM code values needed for direct import into the ITSFM database.
MicroStation this export outputs MicroStation Code values used for direct import into plan sheets.
ESRI Shapefile this export outputs feature and attribute values needed in ESRI GIS systems.
Google KML this export outputs features to Google Earth KML files.

CSV Export File

For advanced users, additional export functionality is available to manipulate data and photograph output. The export routines can: 1) Perform code list lookup to output values over 10 characters (i.e. many MicroStation and street names exceed 10 characters); 2) Populate the Site Id Name field by combining values from other fields (i.e. device type, route, nearest 1/10 mile marker, nearest travel lane, and unique identifier; 3) Create output folders named with the Site Id Name; 4) Rename photographs based on data dictionary description; and 5) Export renamed photos to the appropriate folder based on Site Id Name.

The Trimble GPS export routines are available to contractors and consultants to assist in preparing as-built deliverables. The routines are provided to users at completion of ITSFM GPS training provided by the Department.
