Feature Import Templates

The FDOT Traffic Engineering and Operations Office keeps Florida safely on the go. For example, the ITSFM software enables accurate data and camera feeds to the statewide 511 Traffic Information System, used by millions of people each year. ITSFM helps keep FDOT’s real-time information accurate.

Introduction

The ITSFM provides for bulk importing as-built information directly into the database using specially designed Feature Import Templates. The templates require Microsoft Excel and provide drop-down menus synchronized to the ITSFM database increase quality and simplify the data input process. The import templates serve as a great tool for transferring data from various as-built data sources into the ITSFM.

The import template provide a lookup of code values necessary to import data into the ITSFM database. Users populate the “Description Tab” sheet (Figure 1), and the import template auto-populates the companion “Code Tab” sheet (Figure 2) containing the code values.
